# Hypothesis Generation Network

## Overview
This skill aids in the scientific reasoning process by proposing hypotheses based on data and designing experiments to test them.

## Capabilities
- **Design Control Experiments**: Suggest appropriate negative and positive controls.
- **Reconstruct Plasmid Pathway**: Plan modifications to existing plasmids (e.g., gene deletion or insertion).

## Usage
- "Suggest a control experiment to verify if the observed fluorescence is due to the new promoter."
- "How should we modify the plasmid to remove the RhlB gene?"

## Example: Control Design
**Goal**: Verify promoter activity.
**Proposal**:
1. **Negative Control**: Promoter-less plasmid (background fluorescence).
2. **Positive Control**: Known constitutive promoter (e.g., J23100).
3. **Experimental**: Test promoter.
