# KEGG Database Access

## Overview
This skill provides access to the Kyoto Encyclopedia of Genes and Genomes (KEGG) for metabolic pathway analysis.

## Capabilities
- **Query KEGG Pathway**: Get details about specific metabolic pathways.
- **Query KEGG Reaction**: Retrieve reaction equations and enzyme codes.

## Usage
- "Find the KEGG pathway for glycolysis in E. coli."
- "What is the reaction catalyzed by EC 1.1.1.1?"

## Example: Pathway Query
**Input**: "map00010" (Glycolysis / Gluconeogenesis)
**Output**: List of genes and compounds involved in the pathway.
