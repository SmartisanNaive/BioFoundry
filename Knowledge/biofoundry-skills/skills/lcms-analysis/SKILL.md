# LC-MS Analysis

## Overview
This skill handles the processing and analysis of LC-MS data for metabolomics.

## Capabilities
- **Process LC-MS Data**: Raw data conversion and preprocessing.
- **Quantify Peaks**: Integrate peak areas for specific m/z values.

## Usage
- "Process the .raw files from the LC-MS run."
- "Quantify the peak for citrate in sample 5."

## Example: Peak Integration
Identify peak retention time and integrate area under the curve.
