# Momentum Workflow

## Overview
This skill manages complex, multi-step experimental campaigns on the Momentum platform, coordinating multiple instruments and protocols.

## Capabilities
- **Design Workflow**: Create a sequence of operations (e.g., Culturing -> Induction -> Lysis -> Assay).
- **Execute Workflow**: Run the designed workflow.

## Usage
- "Design a workflow for a 3-day evolution experiment."
- "Execute the 'Screening_Workflow_v2'."

## Example: Workflow Definition
1. Dispense Media (Multidrop)
2. Inoculate (Colony Picker)
3. Incubate 18h (Incubator)
4. Read OD600 (Plate Reader)
